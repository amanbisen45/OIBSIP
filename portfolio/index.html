My Projects

Project 1

(Temperature Convertert)

You will see a temperature converter interface with an input field for the temperature and a dropdown menu to select the unit (Celsius or Fahrenheit). When you click the "Convert" button, it will display the converted temperature below. The CSS styles the elements to center them on the page and gives a basic layout. The JavaScript function convertTemperature() gets the temperature value and the selected unit from the input fields. It performs the conversion based on the selected unit and updates the result element with the converted temperature. The result is displayed in the result paragraph element.

Project 2

(Word Counter)

You will see a textarea where you can enter text, and when you click the "Count Words" button, it will display the word count below the button. The CSS styles the elements to center them on the page and gives a basic layout. The JavaScript function countWords() gets the text from the textarea, trims any leading/trailing whitespace, splits the text into words using a regular expression (/\s+/), and then counts the number of words. The result is displayed in the wordCount span element.
